Routing Quiz

Splendid!

Instructions

  • add ui-router to the project
  • setup app to have the following URL state:
    • / (default state) loads the instructions
    • /bricks/red loads the red bricks template
    • /bricks/blue loads the blue bricks template
    • /bricks/green loads the green bricks template
    • /cart loads the cart template
  • add the following links to the nav:
    • Home (/) with ID homeLink
    • Red (/bricks/red) with ID redBrickLink
    • Blue (/bricks/blue) with ID blueBrickLink
    • Green (/bricks/green) with ID greenBrickLink
    • Cart (/cart) with ID cartLink
  • add a nested state to each brick that loads the cart template (e.g. /bricks/red/cart)
